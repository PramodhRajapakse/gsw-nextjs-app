# Golden State Warriors Website

![Golden State Warriors](https://img.shields.io/badge/NBA-Golden%20State%20Warriors-1D428A)
![Next.js](https://img.shields.io/badge/Next.js-14.0.0-black)
![React](https://img.shields.io/badge/React-18.0.0-blue)
![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-3.3.0-38B2AC)

A modern, responsive website for the Golden State Warriors basketball team built with Next.js, React, and Tailwind CSS. This project showcases the team, players, schedule, news, and more in the iconic Warriors blue and gold colors.

![Website Preview](https://via.placeholder.com/800x400?text=Golden+State+Warriors+Website)

## Features

- **Responsive Design**: Fully responsive layout that works on mobile, tablet, and desktop devices
- **Modern UI**: Clean, modern interface using the Warriors' brand colors
- **Multiple Pages**:
  - Home page with hero section, upcoming games, news, and team highlights
  - Team roster page with player cards and coaching staff
  - Schedule page with upcoming and past games
  - News section with articles and newsletter signup
- **Interactive Components**: Tabs, cards, and interactive UI elements
- **Performance Optimized**: Built with Next.js for optimal performance and SEO

## Technologies Used

- **Next.js**: React framework for server-side rendering and static site generation
- **React**: JavaScript library for building user interfaces
- **Tailwind CSS**: Utility-first CSS framework
- **shadcn/ui**: High-quality UI components built with Radix UI and Tailwind CSS
- **TypeScript**: Type-safe JavaScript
